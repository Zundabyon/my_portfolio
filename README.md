# 🎮 めたっぴの冒険の書

SFC時代のRPG「冒険の書」をモチーフに制作した、エンジニアポートフォリオです。

ゲームの世界観を取り入れながら、プロフィール・制作物・技術スタック・学習記録を閲覧できるようにしています。

## 💡 制作背景

RUNTEQで学習を進める中で、

「自分らしいポートフォリオを作りたい」

という思いから制作しました。

私はSFC時代のRPGが好きで、プロフィールを見るだけではなく
「ゲームを遊ぶ感覚」で見てもらえるポートフォリオを目指しています。
